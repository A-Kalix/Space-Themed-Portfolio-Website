# Portfolio Website

Personal portfolio built with Next.js 14 and Tailwind CSS.

## Overview
Currently in active development. The goal is to build a high-performance portfolio to showcase my full-stack engineering projects. I'm focusing on setting up a solid architecture first with a clean Tailwind design system before adding more complex components.

## Tech Stack
- Next.js 14 (App Router)
- Tailwind CSS
- Vercel for deployment

## Local Setup
1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`
