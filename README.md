# Portfolio Website

Personal portfolio built with Next.js 14, React Three Fiber, and Tailwind CSS.

**Live Demo:** [https://space-themed-portfolio-website-84djd1p7y.vercel.app](https://space-themed-portfolio-website-84djd1p7y.vercel.app)

## Overview
A high-performance, space-themed portfolio designed to showcase my full-stack engineering projects. Built with a modern Next.js App Router architecture, a clean Tailwind design system, and dynamic Framer Motion animations.

## Tech Stack
- Next.js 14 (App Router)
- React Three Fiber / Drei
- Tailwind CSS
- Framer Motion
- Vercel for deployment

## Local Setup
1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`
